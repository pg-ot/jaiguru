# Jaigurugiri Yogashaala

**The Saffronic Path to Healing**

A premium wellness platform offering holistic transformation through the ancient wisdom of yoga, conscious food discipline, and meditation.

## About

Jaigurugiri Yogashaala was founded by Guru Sivaganesan after two decades of personal transformation. Born from lived experience with heart ailments and chronic illness, the methodology combines three foundational pillars:

- **Right Food** - Conscious eating and nutritional discipline
- **Yoga** - Physical strength and movement practices
- **Meditation** - Mental clarity and emotional balance

## Features

- Responsive single-page application
- 21-Day Reset wellness program
- Global mentorship and online sessions
- Community workshops and guided paths
- Mobile-first design with smooth interactions

## Technology Stack

- HTML5 with semantic markup
- Tailwind CSS for styling
- Vanilla JavaScript for interactions
- Google Fonts (Cinzel, EB Garamond, Playfair Display)
- Schema.org structured data for SEO

## Setup

1. Clone the repository
2. Open `page.html` in a modern browser
3. No build process required - pure HTML/CSS/JS
